Overview

This tool allows you to create a clustered graph with lots of control over it's structure. You configure generators to generate clusters from given node degree distributions. Then, you configure how these clusters are interconnected. You can make use of cluster groups and replication to quickly build a hierarchical structure of many clusters. You can also specifiy the distribution of numerical node attributes on the graph and view various graph measures. Finally, you can simulate an information diffusion process on the graph and export the graph or the configuration that generates it.



User Interface

Cluster List

The cluster list provides an overview of the clusters. It allows to browse the hierarchical structure and to select, to add and to delete clusters.

Graph Visualizations

There are two visualizations for the connections between clusters: The adjacency matrix and the node-link diagram. These visualizations can show different levels of the cluster hierarchy. The node-link diagram can show the additional result level "R", i.e. the graph resulting from the cluster structure. In the matrix, the diagonal shows the edge count within a cluster.

There are additional options for the node-link diagram: Circular layout arranges all nodes around the currently focused cluster and hides connections between the other clusters. This allows for overlap-free editing of the connections. Node coloring and edge coloring apply the cluster color on the result level. Node size scales the node radius with the cluster size or with the node degree on the result level. Highlighting selected edges fades out other edges to aid with identifying selected edges on the result level. Node ratio narrows one end of an edge to represent the ratio of nodes eligible for the connection from each cluster.

Tabs

Cluster Properties: Configure a generator or group that defines a cluster. Replication factors higher than 1 create copies of the cluster definition.

Cluster Connections: Configure each of the selected connections by setting edge count, node ratio, degree assortativity and degree distributions.

Graph Measures: Measures of the selected cluster or whole graph.

Layout Settings: Tuning the force-directed layout of the resulting graph. Sampling helps with performance and overlap, higher gravity leads to more equally-spaced nodes, additoinaly layout iterations can help with unfolding.

Diffusion Simulation: simulate information diffusion processes with manually placed seed nodes.

Import/Export: Import or export a configuration or export the resulting graph.

Controls

Adjacency Matrix

Shift + Scroll: Go through levels
Click: Select connection (on diagonal to select all)
Shift+ Click: Toggle connection direction

Node-Link Diagram

Shift + Scroll: Go through levels
Scroll: Zoom
Drag: Pan view
Click: Select cluster
Right-click: Select connections (on focused cluster to select all)
Shift + right-click: Toggle connection direction
