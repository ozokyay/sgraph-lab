CluNeD


Overview

This tool allows you to create a clustered network with lots of control over its structure. You configure generators to generate clusters from edge counts or node degree distributions. Then, you configure how these clusters are interconnected. You can make use of cluster groups and replication to quickly build a hierarchical structure of many clusters. Various graph statistics help you gain insight into network properties. Finally, you can simulate an information diffusion process on the graph and export the graph or the configuration that generates it.



User interface

Cluster list

The cluster list provides an overview of the clusters. It allows to browse the hierarchical structure and to select, to add and to delete clusters. You can also set the visibility of clusters.

Network visualizations

There are two visualizations for the connections between clusters: The node-link diagram and the adjacency matrix. These visualizations can show different levels of the cluster hierarchy. The node-link diagram can show the network at the node level "N". In the matrix, cells show the edge counts of connections between clusters. Cells on the diagonal show the edge count within a cluster.

The toolbar in the upper left corner provides options to configure the node-link diagram.

  • The node radius can be scaled with the cluster size or with the node degree at the result level.
  • Cluster labels can be shown in the visualization.
  • Egocentric layout arranges all nodes around the currently selected cluster and hides connections between the other clusters (cluster level only).
  • The ratio of the node fractions of cluster connections can be encoded in the widths of edge endpoints (cluster level only).
  • Node coloring and edge coloring apply the cluster color to nodes and edges (node level only).

Tabs

Cluster Properties: Configure a generator or group that defines a cluster. Replication factors higher than 1 create more instances with the same parameters. Degree distributions can be generated with a curve generator, drawn in the line chart, and by editing the list of points.

Cluster Connections: Configure the selected connections by setting edge count, node sampling fractions, degree assortativity and degree distributions.

Graph Measures: Measures of the selected cluster or entire graph.

Layout Settings: Tuning the force-directed layout of the resulting graph. Sampling helps with performance and overlap, higher gravity leads to more equally-spaced nodes and prevents stretching of sparse clusters, and more layout iterations can help with unfolding of large networks.

Diffusion Simulation: Simulate information diffusion processes with manually placed seed nodes.

Import/Export: Import or export a configuration or export the graph.

Controls

Node-Link diagram

Shift + Scroll: Go through levels
Scroll: Zoom
Drag: Pan view
Click: Select cluster
Right-click if a node is selected: Select/deselect connection
Shift + right-click: Swap connection endpoints
Drag from node: Select new connection
Click on edge: Toggle selection
Right-click on edge: Swap connection endpoints

Adjacency matrix

Shift + Scroll: Go through levels
Click: Select connection (on diagonal to select all)
Shift + Click: Toggle connection direction

Line Chart

Click: Create point or set point position
Right-click: Delete a point
Drag: Move a point
